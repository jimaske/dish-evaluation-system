#### 菜品评比小程序
##### 包含功能
- 展示所有比赛列表信息
- 展示某个比赛详细信息
- 点赞比赛评论
- 收藏某个比赛
- 投票某个比赛的作品
- 评论某个比赛或者作品
- 注册登录
- 创建比赛（主办方）
- 参加比赛（参赛方）
- 修改个人资料
